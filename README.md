# GD BetterLoot
Improves vanilla loot generated in structures.

![GD BetterLoot - Preview](https://static.wixstatic.com/media/31958c_7e8bfcb6e7be4ff6a86ff91ba95a2493~mv2.png)

## Supported structures
- Ancient City
- Desert Pyramid
- Igloo
- Jungle Temple
- Mineshaft
- Pillager Outpost
- Ruined Portal
